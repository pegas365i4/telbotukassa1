# telbotkassa1
# Как подключить онлайн кассу ЮКасса к Telegram боту на aiogram
Бот тестирует платежи через Юкассу

## Описание:
Сделано по уроку.
Использовать виртуальное окружение.

## Install pip:
`pip install aiogram`

## Files:
main.py - запускной файл телеграмм бота

markups.py - меню бота
